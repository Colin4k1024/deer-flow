# 前沿制造领域深度研究 Skill

## 适用场景

当用户要求研究以下主题时激活此 skill：
- 协作机器人 / 人机协作
- 智能工厂 / 工业AI / 工业4.0
- 3D打印 / 增材制造
- 先进制造技术
- 智能制造 / 数字化制造

## 研究框架

### Phase 1: 领域扫描

从宽泛到具体确定研究范围：

```
初始搜索：
- "[具体主题] 最新进展 2025 2026"
- "[主题] 技术突破"
- "[主题] 市场规模"

识别维度：
- 技术维度：核心技术、关键突破、技术路线
- 市场维度：市场规模、增速、竞争格局
- 应用维度：落地场景、案例、效果
- 趋势维度：短期、中期、长期趋势
```

### Phase 2: 深度挖掘

按维度进行针对性研究：

#### 技术维度搜索
```
- "[主题] 技术突破 2025"
- "[主题] 专利 最新"
- "[主题] 论文 研究"
- "collaborative robot AI integration 2025"
- "industrial AI large language model manufacturing"
- "additive manufacturing metal 2025"
```

#### 市场维度搜索
```
- "[主题] 市场规模 2025"
- "[主题] 市场报告 2026"
- "collaborative robot market size 2025"
- "smart factory market forecast 2026"
- "3D printing industry report 2025"
```

#### 应用维度搜索
```
- "[主题] 应用案例"
- "[主题] 落地场景"
- "use case [topic] manufacturing"
- "case study [topic] 2025"
```

#### 数据来源优先级
1. **权威机构**：GGII、Grand View Research、IDTechEx、Wohlers Report
2. **行业媒体**：IEEE Robotics、Robotics Business Review、3D Printing Industry
3. **企业财报**：ABB、Fanuc、KUKA、Universal Robots、Stratasys、3D Systems
4. **政府/协会**：工信部、中国机器人产业联盟、A3 Association
5. **学术论文**：ScienceDirect、IEEE Xplore、Google Scholar

### Phase 3: 验证与交叉

确保信息可靠性的检查清单：

- [ ] 主要数据点是否来自2个以上独立来源？
- [ ] 数据是否有具体时间戳（2024/2025/2026）？
- [ ] 市场预测是否标注了预测机构和预测方法？
- [ ] 技术评估是否区分了"已成熟"和"实验性"？
- [ ] 中国市场数据和国际数据是否分开统计？

### Phase 4: 综合输出

#### 资料汇编格式
```markdown
## [主题分类]

### 来源1：标题
- **来源**：机构/媒体名称
- **发布时间**：YYYY-MM
- **链接**：URL
- **摘要**：100-200字内容摘要
- **关键数据**：具体数字
```

#### 分析报告格式
```markdown
## 一、核心技术发现
## 二、市场与产业动态
## 三、主要玩家与竞争格局
## 四、典型应用案例
## 五、趋势与预测
## 六、核心洞察（5-8个）
```

## 制造领域专业术语

| 英文 | 中文 | 说明 |
|------|------|------|
| Cobot / Collaborative Robot | 协作机器人 | 与人类协同工作的机器人 |
| Smart Factory | 智能工厂 | 高度数字化、自动化的制造设施 |
| Digital Twin | 数字孪生 | 物理实体的虚拟复制 |
| Additive Manufacturing | 增材制造 | 3D打印的正式名称 |
| IIoT | 工业物联网 | Industrial Internet of Things |
| Predictive Maintenance | 预测性维护 | 基于数据分析的设备维护 |
| Hyperautomation | 超自动化 | AI+ML+RPA 的端到端自动化 |
| SLM | 选区熔化 | Selective Laser Melting，金属3D打印技术 |
| DED | 直接能量沉积 | Directed Energy Deposition |
| FDM | 熔融沉积 | Fused Deposition Modeling |

## 时间敏感查询

制造领域研究要特别注意时间，因为：
- 技术迭代快（1-2年就有代际更新）
- 市场数据滞后（2025年数据通常2026年才发布）
- 标准更新频繁（ISO标准通常5年一更新）

**当前时间：2026年**
- 优先使用2024-2026年数据
- 2019-2023年数据作为历史背景
- 避免使用2018年以前的数据作为主要证据

## 输出语言

**全部使用中文输出**
- 所有报告、资料、分析用中文撰写
- 专业术语首次出现时附英文
- 地名、品牌名保持原文
